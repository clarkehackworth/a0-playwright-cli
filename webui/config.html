Browser Model
Configure a dedicated LLM for browser tasks.
Leave Provider and Model empty to fall back to the agent's chat model automatically.
Provider
LLM service provider for browser tasks.
Model Name
Model identifier for the selected provider.
Examples: gpt-4o, claude-3-5-sonnet-20241022, gemini/gemini-2.0-flash, llama3.1:8b
API Key (optional)
Override API key for this provider. Leave empty to use the provider's default key already configured in Agent Zero.
API Base URL (optional)
Custom base URL for the provider endpoint.
Required for self-hosted models, e.g. http://localhost:11434 for Ollama.
Remote Browser
Attach to an already-running Chrome, or have this plugin launch a dedicated one per task. Precedence: CDP Endpoint > Launch Chrome > Headed.
CDP Endpoint (optional)
Connect to Chrome you started yourself instead of launching one.
Start it with: google-chrome --remote-debugging-port=9222, then e.g. http://127.0.0.1:9222
Launch Dedicated Chrome
Launch a separate Chrome (own profile, own window) per task instead of using playwright-cli's own browser.
Headed
Open a visible Chrome window for each browser task. Requires a display.
Pentesting (Burp Suite)
Routes browser traffic through an intercepting proxy and tags requests for correlation in Burp's proxy history. These fields apply only to the remote-browser modes (see the Remote Browser section) — not to the default in-process browser. Scope differs per field: Proxy Server and Ignore Certificate Errors apply only to Launch Dedicated Chrome; PwnFox tagging applies to both Launch Dedicated Chrome and CDP Endpoint (attach) modes.
Proxy Server (optional)
Intercepting proxy address, e.g. Burp's listener. Launch Dedicated Chrome mode only — in CDP-attach mode, pass --proxy-server when you start Chrome yourself.
Ignore Certificate Errors
Skip TLS warnings from Burp's CA. Alternative: install Burp's CA as trusted instead. Launch Dedicated Chrome mode only — in CDP-attach mode, pass --ignore-certificate-errors yourself.
PwnFox-style Headers
Tag every request (including tabs/popups opened later) with an X-PwnFox-Color header (PwnFox convention) so Burp's proxy history can be filtered/highlighted per window. Works in both remote modes (Launch Dedicated Chrome and CDP Endpoint attach). If neither remote mode is on, enabling this forces a dedicated Chrome launch — plain playwright open exposes no CDP port to tag through.
PwnFox Color (optional)
Fix the color used for the header above. Leave on auto to derive a stable color per browser-agent window name, so concurrent windows get distinct colors automatically.
Status
Playwright CLI is active. The browser_agent tool is available to all agents.
If Playwright CLI or Chromium are not yet installed, use the Initialize button (plugin list page) to install them automatically.
